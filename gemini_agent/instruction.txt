Первый запуск / Создать Virtual Environment (версия python 3.10):
python -m venv .venv
.\.venv\Scripts\activate
python.exe -m pip install --upgrade pip
pip install -r requirements.txt

Запустить проект командами:
.\.venv\Scripts\activate
rasa run actions
rasa shell

После запуска rasa shell ввести команду:
/session_start
Выбрать язык стрелками
